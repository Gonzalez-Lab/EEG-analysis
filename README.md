EEG-analysis

Pipeline simple de análisis de EEG y clasificación automática de potenciales evocados (ERP) usando Python y Machine Learning.

Repositorio: EEG-analysis

¿Qué hace este proyecto?

Este repositorio implementa un flujo completo de análisis de EEG para distintos paradigmas cognitivos:

Filtrado temporal
Extracción de épocas (epoching)
Corrección de baseline
Rechazo automático de artefactos
Generación de ERPs
Extracción de features
Clasificación con Machine Learning
Validación cruzada (Stratified K-Fold, support vector machine y random forest)

El objetivo es explorar si señales EEG obtenidas EEG multichannel permiten detectar patrones asociados a:

Atención
Lenguaje
Emoción
Conflicto cognitivo
Aprendizaje por recompensa
